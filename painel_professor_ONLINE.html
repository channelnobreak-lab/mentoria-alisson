🍎 Configurar publicação para Apple

Para gerar links que abrem diretamente no Safari, você precisa de uma conta gratuita no Netlify e um token de acesso.

Como obter o token (1 vez só):
1. Acesse app.netlify.com e crie uma conta gratuita
2. Vá em User Settings → Applications → Personal access tokens
3. Clique em New access token, dê um nome (ex: "Mentoria") e copie o token
